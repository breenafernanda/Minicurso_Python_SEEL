Desafios Interativos

Pratique seus conhecimentos com desafios de programação. Escreva seu código e veja a solução quando precisar de ajuda.

Como Funcionam os Desafios

Cada desafio apresenta um problema de programação relacionado aos tópicos do minicurso. Você deve escrever o código Python para resolver o problema.

  • Botão Ver Solução: Mostra uma solução possível para o problema.
  • Botão Reiniciar: Limpa seu código e volta ao estado inicial.

Os desafios são uma ótima maneira de praticar e fixar o conhecimento adquirido durante o minicurso.

Desafio 1: Olá, Mundo!

Escreva um programa que imprima a mensagem "Olá, mundo!" na tela.

Este é um desafio simples para praticar a sintaxe básica do Python.

Desafio 2: Verificação de Maioridade

Crie uma função chamada verificar_maioridade que recebe a idade como parâmetro e retorna uma mensagem indicando se a pessoa é maior ou menor de idade.

A função deve retornar "Você é maior de idade." se a idade for maior ou igual a 18, e "Você é menor de idade." caso contrário.

Desafio 3: Automação com Selenium

Escreva um código que use o Selenium para:

  1. Inicializar o Chrome WebDriver
  2. Acessar uma URL
  3. Encontrar um elemento na página
  4. Fechar o navegador

Desafio 4: Análise de Dados com Pandas

Escreva um código que use o Pandas para:

  1. Ler um arquivo Excel ou CSV
  2. Exibir as primeiras linhas do DataFrame
  3. Realizar uma análise básica dos dados

Desafio 5: Manipulação de Planilhas com Openpyxl

Escreva um código que use o Openpyxl para:

  1. Criar um novo Workbook
  2. Adicionar dados a uma planilha
  3. Salvar o arquivo

Desafio 6: Manipulação de PDF

Escreva um código que use uma biblioteca para manipulação de PDF para:

  1. Criar um novo PDF
  2. Adicionar texto ao PDF
  3. Salvar o arquivo

Desafio 7: Projeto Integrado

Este é um desafio mais complexo que integra várias tecnologias vistas no minicurso. Crie um script que:

  1. Use o Selenium para extrair dados de um site (pode ser simulado)
  2. Armazene os dados em um DataFrame do Pandas
  3. Salve os dados em uma planilha Excel usando Openpyxl
  4. Gere um relatório em PDF com os dados coletados

Este desafio é mais avançado e pode ser adaptado conforme seu nível de conhecimento.

Próximos Passos

Agora que você praticou com os desafios, está pronto para aplicar esses conhecimentos em seus próprios projetos de automação.

Lembre-se de que a prática é fundamental para o aprendizado. Continue experimentando com diferentes cenários e problemas para aprimorar suas habilidades.

Se tiver dúvidas ou precisar de ajuda, não hesite em entrar em contato!

Introdução ao Python

Aprenda os conceitos básicos da linguagem Python, uma das mais populares e versáteis do mundo da programação.

Estrutura Básica do Python

Python é uma linguagem muito intuitiva. Vamos começar com os fundamentos: variáveis, tipos de dados e controle de fluxo.

Variáveis

Variáveis são usadas para armazenar dados que podemos reutilizar. Em Python, não é necessário declarar o tipo da variável, pois a linguagem faz isso automaticamente.

Exemplo de Variáveis
nome = "Mario"
idade = 23
print(f"Olá, meu nome é {nome} e eu tenho {idade} anos.")

No exemplo acima, criamos duas variáveis: nome (que armazena um texto) e idade (que armazena um número). Em seguida, usamos a função print() com uma f-string para exibir essas variáveis.

Tipos de dados

Os principais tipos de dados em Python são:

Tipos de Dados Básicos
numero = 10          # int (número inteiro)
pi = 3.14            # float (número decimal)
texto = "Automação"  # str (texto/string)
verdadeiro = True    # bool (booleano: True ou False)

Python também possui tipos de dados mais complexos como listas, tuplas, dicionários e conjuntos, que veremos mais adiante.

Controle de fluxo

O controle de fluxo permite que seu programa tome decisões com base em condições.

Estrutura Condicional (if-else)
idade = 23
if idade >= 18:
    print("Você é maior de idade.")
else:
    print("Você é menor de idade.")

No exemplo acima, o programa verifica se a idade é maior ou igual a 18. Se for, exibe uma mensagem; caso contrário, exibe outra.

Funções e Boas Práticas

Funções são blocos de código reutilizáveis que nos ajudam a organizar nossos programas.

Definição de funções

Em Python, usamos a palavra-chave def para definir uma função:

Definindo uma Função
def saudacao(nome):
    texto = (
        f"Olá, {nome}! "
        f"Bem-vindo ao minicurso."
    )
    return texto

nome = "Bob"
texto_de_saudacao = saudacao(nome)
print(texto_de_saudacao)

A função saudacao recebe um parâmetro nome e retorna um texto personalizado. Observe como podemos quebrar strings longas em várias linhas para melhor legibilidade.

Boas práticas

  • Nomes descritivos: Use nomes de variáveis e funções claros e descritivos.
  • Comentários: Comente seu código onde necessário para explicar partes complexas.
  • Organização: Organize seu código em funções para torná-lo mais legível e reutilizável.
  • Indentação: Python usa indentação (espaços ou tabs) para definir blocos de código, então seja consistente.

Exemplo Prático

Vamos juntar tudo em um exemplo prático que utiliza entrada de dados, controle de fluxo e funções.

Verificação de Maioridade
# Função para verificar maioridade
def verificar_maioridade(idade):
    if idade >= 18:
        return "Você é maior de idade."
    else:
        return "Você é menor de idade."

# Definindo variáveis
nome = input("Digite seu nome: ")
idade = int(input("Digite sua idade: "))

# Resultado
print(f"Olá, {nome}!")
print(verificar_maioridade(idade))

Neste exemplo:

  1. Definimos uma função verificar_maioridade que recebe uma idade e retorna uma mensagem.
  2. Usamos a função input() para obter o nome e a idade do usuário.
  3. Convertemos a idade para um número inteiro com int().
  4. Exibimos uma saudação personalizada e o resultado da verificação de maioridade.

Este é um exemplo simples, mas demonstra como podemos combinar diferentes conceitos para criar programas úteis.

Estruturas de Dados

Python possui várias estruturas de dados embutidas que são muito úteis para diferentes situações.

Listas

Listas são coleções ordenadas e mutáveis de itens. Podem conter itens de diferentes tipos.

Trabalhando com Listas
# Criando uma lista
frutas = ["maçã", "banana", "laranja"]

# Acessando elementos
primeira_fruta = frutas[0]  # maçã

# Modificando elementos
frutas[1] = "morango"  # Agora a lista é ["maçã", "morango", "laranja"]

# Adicionando elementos
frutas.append("uva")  # Adiciona ao final: ["maçã", "morango", "laranja", "uva"]

# Removendo elementos
frutas.remove("maçã")  # Remove a maçã: ["morango", "laranja", "uva"]

# Iterando sobre uma lista
for fruta in frutas:
    print(fruta)

Dicionários

Dicionários são coleções de pares chave-valor, onde cada chave é única.

Trabalhando com Dicionários
# Criando um dicionário
pessoa = {
    "nome": "Carlos",
    "idade": 30,
    "profissao": "Engenheiro"
}

# Acessando valores
nome = pessoa["nome"]  # Carlos

# Modificando valores
pessoa["idade"] = 31

# Adicionando novos pares
pessoa["cidade"] = "São Paulo"

# Verificando se uma chave existe
if "profissao" in pessoa:
    print(f"Profissão: {pessoa['profissao']}")

# Iterando sobre um dicionário
for chave, valor in pessoa.items():
    print(f"{chave}: {valor}")

Loops

Python oferece dois tipos principais de loops: for e while.

Loop for

O loop for é usado para iterar sobre uma sequência (como lista, tupla, dicionário, conjunto ou string).

Exemplos de Loop for
# Iterando sobre uma lista
numeros = [1, 2, 3, 4, 5]
for numero in numeros:
    print(numero)

# Iterando sobre um range
for i in range(5):  # 0, 1, 2, 3, 4
    print(i)

# Iterando sobre uma string
for letra in "Python":
    print(letra)

Loop while

O loop while executa um bloco de código enquanto uma condição for verdadeira.

Exemplos de Loop while
# Contagem regressiva
contador = 5
while contador > 0:
    print(contador)
    contador -= 1  # Equivalente a: contador = contador - 1

# Loop com break
while True:
    resposta = input("Digite 'sair' para encerrar: ")
    if resposta.lower() == "sair":
        break  # Sai do loop
    print(f"Você digitou: {resposta}")

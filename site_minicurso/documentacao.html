Guia completo sobre a estrutura e funcionamento do site do Minicurso de Python.

Estrutura do Site

O site do Minicurso de Python está organizado em várias seções, cada uma focada em um aspecto específico do aprendizado de Python e automação. Abaixo está uma explicação detalhada de cada seção:

1. Página Inicial

A página inicial apresenta uma visão geral do minicurso, destacando os principais tópicos que serão abordados e fornecendo links rápidos para as seções mais importantes.

2. Introdução ao Python

Esta seção cobre os conceitos básicos da linguagem Python, incluindo:

  • Variáveis e tipos de dados
  • Estruturas de controle (if, else, loops)
  • Funções e módulos
  • Estruturas de dados (listas, dicionários, etc.)

3. Automação com Selenium

Aqui você aprenderá a usar o Selenium para automatizar tarefas em navegadores web, incluindo:

  • Configuração do ambiente
  • Navegação em páginas web
  • Localização e interação com elementos
  • Extração de dados

4. Análise com Pandas

Esta seção ensina como usar a biblioteca Pandas para análise de dados, abordando:

  • Criação e manipulação de DataFrames
  • Filtragem e seleção de dados
  • Agregação e estatísticas
  • Visualização básica

5. Planilhas com Openpyxl

Aprenda a manipular arquivos Excel usando a biblioteca Openpyxl:

  • Leitura e escrita de planilhas
  • Formatação de células
  • Criação de fórmulas
  • Geração de relatórios

6. Manipulação de PDF

Esta seção mostra como criar e manipular arquivos PDF com Python:

  • Criação de documentos PDF
  • Adição de texto e imagens
  • Extração de conteúdo de PDFs existentes
  • Mesclagem e divisão de arquivos

7. Desafios Interativos

Pratique seus conhecimentos com desafios de programação interativos, onde você pode:

  • Escrever código Python diretamente no navegador
  • Ver soluções para os desafios
  • Reiniciar os desafios para tentar novamente

8. Resoluções Beecrowd

Acesse mais de 200 soluções de problemas da plataforma Beecrowd (antigo URI Online Judge), organizadas por número e nome do problema.

Como Usar o Site

Navegação

O menu lateral permite navegar facilmente entre as diferentes seções do site. Cada item do menu é identificado por um número e um título, facilitando a localização do conteúdo desejado.

Exemplos de Código

Os exemplos de código são apresentados em blocos destacados com sintaxe colorida para facilitar a leitura. Você pode copiar o código clicando no botão "Copiar" que aparece no canto superior direito de cada bloco.

Desafios Interativos

Na seção de Desafios Interativos, você pode:

  1. Ler a descrição do desafio
  2. Escrever seu código no editor fornecido
  3. Clicar em "Ver Solução" para ver uma solução possível
  4. Clicar em "Reiniciar" para limpar o editor e começar novamente

Resoluções Beecrowd

Na seção de Resoluções Beecrowd, você pode:

  1. Pesquisar problemas por número ou nome usando a barra de pesquisa
  2. Clicar em "Ver Solução" para visualizar o código da solução
  3. Copiar o código para usar como referência

Estrutura Técnica do Site

Para aqueles interessados em entender como o site foi construído, aqui está uma visão geral da estrutura técnica:

Tecnologias Utilizadas

  • HTML5: Para a estrutura das páginas
  • CSS3: Para o estilo e layout
  • JavaScript: Para interatividade e funcionalidades dinâmicas
  • CodeMirror: Para o editor de código nos desafios interativos
  • Font Awesome: Para ícones
  • Google Fonts: Para tipografia

Organização de Arquivos

O site está organizado da seguinte forma:

  • Arquivos HTML: Um para cada seção principal do site
  • CSS: Estilos globais em style.css e estilos específicos para destaque de sintaxe em syntax-highlight.css
  • JavaScript: Funcionalidades gerais em script.js e destaque de sintaxe em syntax-highlight.js
  • Imagens: Armazenadas na pasta img/
  • Código Beecrowd: Soluções armazenadas na pasta beecrowd_code/

Implementando Novas Funcionalidades

Se você deseja adicionar novas funcionalidades ao site, aqui estão algumas orientações:

Adicionando Novos Módulos

  1. Crie um novo arquivo HTML baseado na estrutura dos existentes
  2. Adicione um novo item ao menu lateral em todos os arquivos HTML
  3. Atualize os números dos itens do menu se necessário
  4. Adicione o conteúdo específico do módulo

Adicionando Novos Desafios

  1. Copie a estrutura HTML de um desafio existente em desafios.html
  2. Atualize os IDs, títulos e descrições
  3. Adicione a solução ao objeto solutions no arquivo script.js

Adicionando Novas Resoluções Beecrowd

  1. Adicione o arquivo Python à pasta beecrowd_code/
  2. Atualize o arquivo beecrowd_problems.json com os metadados do novo problema

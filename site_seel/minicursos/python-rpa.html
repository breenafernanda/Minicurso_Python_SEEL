Sobre o Minicurso

O minicurso "Introdução ao Python para RPA" foi desenvolvido para estudantes de Engenharia Elétrica que desejam aprender a automatizar tarefas repetitivas utilizando Python. RPA (Robotic Process Automation) é uma tecnologia que permite automatizar processos baseados em regras, reduzindo o tempo gasto em tarefas manuais e aumentando a produtividade.

Durante o minicurso, os participantes aprenderão os fundamentos da linguagem Python e como utilizar bibliotecas específicas para automação, como Selenium para automação web, Pandas para manipulação de dados, Openpyxl para trabalhar com planilhas Excel e PyPDF2 para manipulação de arquivos PDF.

Não é necessário conhecimento prévio em programação, mas familiaridade com lógica de programação será útil. Todos os participantes devem trazer seus próprios laptops com Python instalado (instruções de instalação serão enviadas previamente).

2. Automação com Selenium

Aprenda a automatizar navegadores web utilizando a biblioteca Selenium, permitindo interagir com páginas web, preencher formulários e extrair dados.

  • Configuração do ambiente
  • Navegação em páginas web
  • Localização e interação com elementos
  • Extração de dados
Exemplo: Automação com Selenium

from selenium import webdriver
from selenium.webdriver.chrome.options import Options
from selenium.webdriver.common.by import By
import time

# Inicializar o navegador Chrome
driver = webdriver.Chrome(Options())

# Acessar uma URL
driver.get("https://www.google.com")

# Encontrar um elemento na página (campo de pesquisa)
campo_pesquisa = driver.find_element(By.NAME, "q")

# Interagir com o elemento
campo_pesquisa.send_keys("Selenium Python")

# Esperar um pouco para visualizar
time.sleep(2)

# Fechar o navegador
driver.quit()

3. Análise de Dados com Pandas

Utilize a biblioteca Pandas para manipular, analisar e visualizar dados de forma eficiente.

  • Criação e manipulação de DataFrames
  • Filtragem e seleção de dados
  • Agregação e estatísticas
  • Visualização básica
Exemplo: Análise com Pandas

import pandas as pd

# Criar um DataFrame de exemplo
dados = {
    'Nome': ['Ana', 'Bruno', 'Carlos', 'Daniela'],
    'Idade': [25, 30, 22, 28],
    'Salário': [3500, 4200, 2800, 5100]
}

# Criar o DataFrame
df = pd.DataFrame(dados)

# Exibir as primeiras linhas
print("Primeiras linhas do DataFrame:")
print(df.head())

# Análise básica
print("\nInformações estatísticas:")
print(df.describe())

# Média de salário por idade
print("\nMédia de salário:")
print(df['Salário'].mean())

4. Manipulação de Planilhas com Openpyxl

Aprenda a manipular arquivos Excel usando a biblioteca Openpyxl, permitindo ler, escrever e formatar planilhas.

  • Leitura e escrita de planilhas
  • Formatação de células
  • Criação de fórmulas
  • Geração de relatórios
Exemplo: Manipulação de Excel

from openpyxl import Workbook
from openpyxl.styles import Font, Alignment, PatternFill

# Criar um novo Workbook
wb = Workbook()
ws = wb.active
ws.title = "Relatório"

# Adicionar cabeçalhos
headers = ["Nome", "Departamento", "Salário"]
for col_num, header in enumerate(headers, 1):
    cell = ws.cell(row=1, column=col_num)
    cell.value = header
    cell.font = Font(bold=True)
    cell.alignment = Alignment(horizontal='center')
    cell.fill = PatternFill(start_color="FFFF00", end_color="FFFF00", fill_type="solid")

# Adicionar dados
data = [
    ["João Silva", "TI", 5000],
    ["Maria Santos", "RH", 4500],
    ["Pedro Alves", "Vendas", 4800]
]

for row_num, row_data in enumerate(data, 2):
    for col_num, cell_value in enumerate(row_data, 1):
        ws.cell(row=row_num, column=col_num).value = cell_value

# Ajustar largura das colunas
for col in ws.columns:
    max_length = 0
    column = col[0].column_letter
    for cell in col:
        if cell.value:
            max_length = max(max_length, len(str(cell.value)))
    adjusted_width = (max_length + 2)
    ws.column_dimensions[column].width = adjusted_width

# Salvar o arquivo
wb.save("relatorio.xlsx")
print("Arquivo Excel criado com sucesso!")

5. Manipulação de PDFs

Aprenda a criar e manipular arquivos PDF com Python, permitindo extrair informações, mesclar documentos e criar relatórios.

  • Criação de documentos PDF
  • Adição de texto e imagens
  • Extração de conteúdo de PDFs existentes
  • Mesclagem e divisão de arquivos
Exemplo: Criação de PDF

from reportlab.lib.pagesizes import letter
from reportlab.pdfgen import canvas
from reportlab.lib import colors
from reportlab.platypus import Paragraph
from reportlab.lib.styles import getSampleStyleSheet

# Criar um novo PDF
c = canvas.Canvas("exemplo.pdf", pagesize=letter)
width, height = letter

# Adicionar um título
c.setFont("Helvetica-Bold", 16)
c.drawString(50, height - 50, "Relatório de Exemplo")

# Adicionar uma linha abaixo do título
c.setStrokeColor(colors.blue)
c.line(50, height - 60, width - 50, height - 60)

# Adicionar texto
c.setFont("Helvetica", 12)
c.drawString(50, height - 100, "Este é um exemplo de PDF criado com Python.")
c.drawString(50, height - 120, "Podemos adicionar texto, imagens e formas.")

# Adicionar uma forma
c.setFillColor(colors.lightblue)
c.rect(50, height - 200, 100, 50, fill=True)

# Adicionar texto dentro da forma
c.setFillColor(colors.black)
c.drawString(70, height - 175, "Python PDF")

# Salvar o PDF
c.save()
print("PDF criado com sucesso!")

6. Projeto Prático Integrado

Neste módulo final, você aplicará todos os conhecimentos adquiridos em um projeto prático que integra as diferentes bibliotecas e técnicas aprendidas durante o minicurso.

O projeto consiste em criar um sistema de automação que:

  1. Utiliza Selenium para extrair dados de um site (por exemplo, produtos de um e-commerce)
  2. Processa os dados extraídos utilizando Pandas
  3. Gera um relatório em Excel com Openpyxl
  4. Cria um resumo em PDF com os principais insights

Projeto Final: Sistema de Automação Integrado

Desenvolva um sistema que automatize a coleta de dados de cotações de moedas, processe esses dados, gere um relatório Excel e um resumo em PDF.

Este projeto integra todas as bibliotecas estudadas no minicurso:

  • Selenium para extrair cotações de um site financeiro
  • Pandas para processar e analisar os dados
  • Openpyxl para gerar um relatório Excel com gráficos
  • ReportLab para criar um resumo em PDF

Instalação do Python

Python Automação

Passo a passo para diferentes sistemas operacionais

Windows

  1. Acesse python.org/downloads
  2. Baixe a versão mais recente (3.10+)
  3. Importante: Marque a opção "Add Python to PATH"
  4. Clique em "Install Now" para instalação padrão
  5. Verifique a instalação abrindo o Prompt de Comando e digitando: 
    python --version

macOS

  1. Acesse python.org/downloads
  2. Baixe a versão mais recente para macOS
  3. Execute o instalador .pkg e siga as instruções
  4. Verifique a instalação abrindo o Terminal e digitando: 
    python3 --version

Linux (Ubuntu/Debian)

  1. Abra o Terminal
  2. Atualize os repositórios: 
    sudo apt update
  3. Instale o Python e o pip: 
    sudo apt install python3 python3-pip
  4. Verifique a instalação: 
    python3 --version
pip3 --version

Dica: Para verificar se o Python foi instalado corretamente, tente executar o interpretador interativo:

python
ou 
python3

Você deverá ver o prompt >>> indicando que o interpretador está funcionando.

Configuração do VS Code

Python Automação

Instalação do VS Code

  1. Acesse code.visualstudio.com
  2. Baixe a versão para seu sistema operacional
  3. Execute o instalador e siga as instruções
  4. Inicie o VS Code após a instalação

Extensões Recomendadas

  • Python (Microsoft): Suporte essencial para Python
  • Pylance: Servidor de linguagem para melhor IntelliSense
  • Python Indent: Melhora a indentação automática
  • Jupyter: Para notebooks interativos
  • Rainbow Brackets: Facilita a visualização de parênteses

Configuração Básica

  1. Abra o VS Code
  2. Acesse a aba de extensões (Ctrl+Shift+X)
  3. Pesquise e instale as extensões recomendadas
  4. Crie uma pasta para o projeto: 
    mkdir automacao_python
cd automacao_python
code .
  5. Crie um novo arquivo Python (File > New File)
  6. Salve como "teste.py" (Ctrl+S)

Atalhos Úteis

  • Ctrl+`: Abrir/fechar terminal integrado
  • F5: Executar o script atual
  • Ctrl+/: Comentar/descomentar linha
  • Ctrl+Space: Ativar sugestões de código

Usando o pip

O pip é o gerenciador de pacotes do Python que permite instalar bibliotecas facilmente.

Instalação das bibliotecas principais:

Configuração do WebDriver

Python Automação

Preparando o Selenium para automação web

O que é o WebDriver?

O WebDriver é uma interface que permite ao Selenium controlar navegadores web. Cada navegador requer seu próprio driver.

Método Simplificado (Recomendado)

Usando o ChromeDriverManager (instalação automática):

pip install webdriver-manager

No seu código:

from selenium import webdriver
from selenium.webdriver.chrome.service import Service
from webdriver_manager.chrome import ChromeDriverManager

driver = webdriver.Chrome(service=Service(ChromeDriverManager().install()))

Método Manual

  1. Verifique a versão do seu navegador Chrome
  2. Baixe o ChromeDriver compatível em: sites.google.com/chromium.org/driver
  3. Extraia o arquivo executável
  4. Adicione ao PATH do sistema ou especifique o caminho no código: 
    from selenium import webdriver

driver = webdriver.Chrome(executable_path='/caminho/para/chromedriver')

Para outros navegadores:

  • Firefox: GeckoDriver
  • Edge: EdgeDriver
  • Safari: SafariDriver

Teste Rápido

Crie um arquivo teste_selenium.py com o seguinte código:

from selenium import webdriver
from selenium.webdriver.chrome.service import Service
from webdriver_manager.chrome import ChromeDriverManager
import time

# Inicializar o driver
driver = webdriver.Chrome(service=Service(ChromeDriverManager().install()))

# Abrir uma página
driver.get('https://www.python.org')

# Aguardar para visualização
time.sleep(3)

# Fechar o navegador
driver.quit()

print("Teste concluído com sucesso!")

Verificação do Ambiente

Python Automação

Confirmando que tudo está funcionando

Script de Verificação

Crie um arquivo verificar_ambiente.py com o seguinte código:

import sys
import pkg_resources
import platform

# Verificar versão do Python
print(f"Python: {platform.python_version()}")

# Verificar bibliotecas instaladas
bibliotecas = [
    "selenium",
    "pandas",
    "openpyxl",
    "fpdf",
    "colorama",
    "webdriver_manager"
]

print("\nBibliotecas instaladas:")
for biblioteca in bibliotecas:
    try:
        versao = pkg_resources.get_distribution(biblioteca).version
        print(f"✓ {biblioteca}: {versao}")
    except pkg_resources.DistributionNotFound:
        print(f"✗ {biblioteca}: Não instalada")

# Testar importação do Selenium
try:
    from selenium import webdriver
    from selenium.webdriver.chrome.service import Service
    from webdriver_manager.chrome import ChromeDriverManager
    print("\n✓ Importação do Selenium bem-sucedida")
except ImportError as e:
    print(f"\n✗ Erro ao importar Selenium: {e}")

print("\nVerificação concluída!")

Problemas Comuns e Soluções

Python não encontrado no PATH

  • Windows: Reinstale o Python marcando "Add Python to PATH"
  • macOS/Linux: Verifique se está usando python3 em vez de python

Erro ao instalar bibliotecas

  • Tente usar --user: pip install --user selenium
  • Verifique a conexão com a internet
  • Atualize o pip: python -m pip install --upgrade pip

Erro com o WebDriver

  • Verifique se o navegador está atualizado
  • Tente usar o webdriver-manager
  • Verifique permissões do arquivo do driver
Se encontrar problemas persistentes, não hesite em pedir ajuda. Configurar o ambiente corretamente é crucial para o sucesso do minicurso.
